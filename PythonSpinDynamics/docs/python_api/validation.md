# Validation

Python functions are validated against small MATLAB or Octave fixtures stored in
`validation/fixtures`.

Generate fixtures from `PythonSpinDynamics`:

```powershell
matlab -batch "run('validation/octave/generate_basic_fixtures.m')"
matlab -batch "run('validation/octave/generate_imaging_fixtures.m')"
matlab -batch "run('validation/octave/generate_optimization_fixtures.m')"
matlab -batch "run('validation/octave/generate_optimization_result_fixtures.m')"
matlab -batch "run('validation/octave/generate_pulse_fixtures.m')"
```

or:

```powershell
& 'C:\Program Files\GNU Octave\Octave-11.3.0\mingw64\bin\octave-cli.exe' --quiet validation\octave\generate_basic_fixtures.m
```

Run tests:

```powershell
python -m unittest discover -s tests
```

The test suite includes example smoke tests for non-plot examples, source-tree
execution from inside `examples/`, and plot-example CLI parsing without
requiring Matplotlib.

The current fixture suite checks:

- echo conversion;
- arbitrary direct-sum echo conversion;
- FID direct-sum trace conversion through the workflow;
- effective-axis helpers;
- rotation matrix elements;
- ideal CPMG asymptotic magnetization;
- ideal CPMG parameter construction;
- ideal FID parameter construction;
- `sim_spin_dynamics_arb10`;
- ideal-probe finite acquisition through `calc_macq_ideal_probe_relax4`;
- tuned, untuned, and matched finite-acquisition receiver wrappers;
- public finite ideal CPMG train acquired spectra, echoes, and echo integrals;
- ideal FID acquired spectrum and trace;
- tuned-probe original/reference transmit current;
- tuned-probe original/reference CPMG asymptotic and received spectra;
- untuned-probe original/reference transmit current;
- untuned-probe original/reference CPMG asymptotic and received spectra;
- matched-probe original/reference matching network and coil current;
- matched-probe original/reference CPMG asymptotic and received spectra;
- ideal, tuned, and matched CPMG imaging k-space arrays;
- JMR rectangular pulse responses, phase quantization, and untuned segment
  adjustment;
- NumPy compatibility helpers used by public workflows;
- example scripts and plot-example command-line interfaces;
- public CPMG workflow result shapes for ideal, tuned, untuned, and matched
  runners.

Matched-probe fixtures require MATLAB toolboxes used by the reference
implementation. Octave fixture generation skips matched-probe files when
`fmincon` is unavailable; the committed matched fixtures are generated by MATLAB.

See `docs/validation_results.md` for the run log.
