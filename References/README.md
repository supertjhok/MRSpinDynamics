# References

This folder is primarily a local source archive for MRSpinDynamics development.
Most of its contents are intentionally ignored because they are published
papers, books, copied reference documents, or large source captures that should
not be redistributed through this repository.

The following self-authored technical notes are tracked because they document
assumptions and derivations used by the public subprojects:

- [Ab Initio Electric-Field Gradients and Quadrupolar Resonances in Crystals](efg_quadrupolar_technical_note.pdf)
  (`efg_quadrupolar_technical_note.tex`) covers electric-field-gradient tensor
  conventions, quadrupolar coupling constants, asymmetry parameters, NQR
  transition frequencies, and practical DFT workflow considerations.
- [Modeling Pulsed NQR Dynamics: Spin 1, Spin 3/2, and Higher Spins](Pulsed_NQR_Spin_Dynamics_Narrative_Rewrite.pdf)
  (`Pulsed_NQR_Spin_Dynamics_Narrative_Rewrite.tex`) explains when pulsed NQR
  simulations can use a reduced two-level model and when they require the full
  density-matrix treatment.
